My projects......

date calculator

这是一个简单的日期计算器，基于 JavaScript 构建，通过不到 200 行代码实现了自动计算两个日期的天数差，解放你的双手。

emoji

通过 setInterval 属性，加入一些你想要的表情符号 ，可以将网页标签栏的 favicon.icon 换成动态的表情，它会一直的播放下去。

page minimap

网页的小地图，就是对整个网页的缩略图，页面中每个有文字或者图片的地方都会被通过深或浅的颜色显示在地图中，和 vs-code 中的小地图类似。

tic-tac-toe

这是一个井字棋游戏，我们小时候经常玩的，在 JavaScript 中的 AI 算法起了很大的作用，有小白或者大师两个级别，它能自动计算难易度和落子位置。Enjoy~

reward

模仿知乎，通过 HTML/CSS/JS 实现了一个简单的网页赞赏模块，由于没有后端和服务器，用了几张收款码实现不同金额收款，也可做到完美的效果。

typewriter

网页打字机，你想要在网站首页逐个打印文字的时候，只需嵌入此模块，将你的文字放在 JavaScript 中固定的地方即可，也可设置打印速度、删除速度等......
